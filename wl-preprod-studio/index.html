↓ Escolha 1 ou 2 directions de mood

Com base no que você criou acima, eu pesquisei o Asset Hub Wildlife e separei 6 directions visuais que funcionam pra esse gênero, todas vindas de jogos reais do portfólio (Farm Day, Paradise Paws). ⚠️
O que essas refs PROMETEM (e o que não): Estas imagens são o norte visual. O Layer (gerador) vai produzir variações no estilo escolhido, mas não vai clonar exatamente os personagens das refs (são IPs Wildlife). Texto dentro de imagem (HUD, logos) sai mal — desenhar fora. Resolução prática: 1024×1024. Pose dinâmica + multidão = falha. Mude expectativas: este mood picker é pra você ESCOLHER A DIREÇÃO, não pra você assumir que o output vai ser 1:1 disso.

Pra escolher

Volta no Claude Code e fala: vou de mood 3, ou misture: mood 3 + 4 (personagens vêm de 3, ícones de 4). Eu gravo isso em style.md com paleta, proporção, shading e don'ts. Aí qualquer comando seguinte ("gera concept", "produz sprite") já roda dentro desse mood — você não precisa repetir nada.

Como o lock funciona daqui pra frente

1. Lock Você escolhe mood 3.
style.md grava: paleta soft beige/lilás, proporção animal chibi, shading painterly soft, don'ts: "sem cores saturadas, sem ação rápida".
→
2. Gerações futuras leem o lock Quando você diz "gera concept do gato chefe", eu construo o prompt automaticamente com os atributos do mood 3 + adiciono o sujeito. Você não repete nada.
→
3. Drift detection Se você pedir algo que contradiz o lock (ex: "agora cyberpunk"), o plugin avisa: "isso conflita com o mood 3, quer (a) atualizar style.md, (b) gerar fora só dessa vez, (c) cancelar".

Como instalar qualquer um destes apps

Cada app é distribuído como plugin de Claude Code. Instalação em 2 passos:

claude plugin marketplace add marcofurtado-hub/wl-preprod-studio
claude plugin install wl-preprod-studio@wildlife-preprod

⚠ Requer acesso aos MCPs internos (Asset Hub, Layer Wildlife) — peça pro time de infra liberar se ainda não tem.
